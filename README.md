# 🚗 Top Climbing

Top Climbing, Unity ile geliştirilmiş 2D fizik tabanlı bir mobil tırmanma oyunudur. Oyuncu aracını kontrol ederek zorlu tepeleri aşmaya çalışır, yakıtını yönetir ve coin toplayarak ilerler.

## 🎮 Oyun Özellikleri

- Gerçekçi araç fiziği
- Yakıt tüketim sistemi
- Coin toplama mekaniği
- Mobil uyumlu kontrol sistemi
- Duraklatma sistemi
- Unity UI kullanımı
- Prefab tabanlı oyun yapısı

## 🛠️ Kullanılan Teknolojiler

- Unity 2022.3 LTS
- C#
- Unity Physics (Rigidbody2D)
- Unity UI System

## 📂 Proje Yapısı

```text
Assets/
├── Art/
├── Prefabs/
├── Scenes/
├── Scripts/
Packages/
ProjectSettings/
```

## 🎯 Kontroller

### Klavye (Editör Testi)

| İşlem | Tuş |
|---------|---------|
| Gaz / İleri | D veya Sağ Ok |
| Fren / Geri | A veya Sol Ok |
| Duraklat | UI Butonu |

### Mobil

| İşlem | Kontrol |
|---------|---------|
| Gaz / İleri | Sağ Alt Buton |
| Fren / Geri | Sol Alt Buton |
| Duraklat | Sağ Üst Buton |

## 🚀 Kurulum

1. Repoyu klonlayın:

```bash
git clone https://github.com/dilaraertugrul/Top-Climbing-MyVersion.git
```

2. Unity Hub ile projeyi açın.

3. Unity 2022.3 LTS sürümünü kullanın.

4. Scriptlerin derlenmesini bekleyin.

5. Oyunu çalıştırmak için Play tuşuna basın.
